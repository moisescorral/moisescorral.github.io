## Tráiler
En `index.html`, busca la sección `trailer-box` y sustituye el bloque `video-placeholder` por el iframe de YouTube.

Ejemplo:

<iframe width="100%" height="100%" src="https://www.youtube.com/embed/TU_ID" title="Tráiler" frameborder="0" allowfullscreen></iframe>
